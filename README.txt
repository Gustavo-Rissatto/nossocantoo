NOSSO CANTINHO - VERSÃO DEFINITIVA ❤️

Essa é a versão para enviar para ela.

O timer está ATIVO:
- O app fica bloqueado antes do próximo dia 04.
- No dia 04, o conteúdo é liberado automaticamente.

Para publicar:
1. Extraia este ZIP.
2. Acesse https://app.netlify.com/drop
3. Arraste a pasta extraída `nosso-cantinho-definitivo`.
4. O Netlify vai gerar um link.
5. Envie o link para ela.

Para instalar no celular:
Android:
- Abrir o link no Chrome
- Tocar em ⋮
- Adicionar à tela inicial

iPhone:
- Abrir o link no Safari
- Compartilhar
- Adicionar à Tela de Início

Para testar sem timer:
No arquivo script.js, troque:
const TIMER_ATIVO = true;
por:
const TIMER_ATIVO = false;
